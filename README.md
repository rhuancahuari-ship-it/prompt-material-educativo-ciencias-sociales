# Prompt para Generación de Material Educativo - Ciencias Sociales (Historia)

## Descripción
Este repositorio contiene un prompt completo y estructurado para generar material educativo de alta calidad para estudiantes de secundaria en el área de Ciencias Sociales, especialidad Historia.

## Características del Prompt

- ✅ Generación de situaciones significativas contextualizadas
- ✅ Desarrollo de temas secuenciales y lógicos
- ✅ Fichas de práctica con lecturas académicas
- ✅ Implementación de Metodologías Activas de Aprendizaje (MAA)
- ✅ Aplicación de principios de neuroeducación y DUA
- ✅ Material dirigido directamente al estudiante

## Estructura del Proceso

El prompt sigue un flujo riguroso de 4 órdenes:

### ORDEN 01: Generación de Temas
- 5 temas secuenciales por competencia
- 2 situaciones significativas por competencia
- Total: 10 temas distribuidos en 2 situaciones

### ORDEN 02: Redacción de Situación Significativa
- 4 párrafos (180-200 palabras totales)
- Reto planteado (2 párrafos, 25-30 palabras c/u)
- Lenguaje cercano y motivador

### ORDEN 03: Ficha de Práctica
- 2 lecturas académicas (250-300 palabras cada una)
- 8 preguntas de alternativa múltiple
- 2 preguntas críticas reflexivas
- Tabla de aleatoriedad para respuestas
- Tabla de justificaciones

### ORDEN 04: Elaboración del Producto
- Uso de MAA: ABP, ABPj o ABI
- Presentación en tabla de 3-4 columnas
- Fases específicas según metodología
- Material para elaboración individual

## Principios Pedagógicos

- **Neuroeducación**: Estimulación de la neuroplasticidad
- **Inteligencias Múltiples**: Enfoque diversificado
- **DUA**: Diseño Universal para el Aprendizaje
- **Redacción directa**: Segunda persona singular o primera plural
- **Progresión lógica**: De lo simple a lo complejo

## Uso del Prompt

1. Especificar grado de secundaria (1° a 5°)
2. Definir competencia a abordar
3. Seguir las órdenes secuencialmente
4. Aprobar cada entrega antes de continuar

## Adaptabilidad

- Compatible con diferentes IA (Gemini, ChatGPT, DeepSeek, etc.)
- Formato Markdown con soporte LaTeX
- Tablas compatibles con Word
- Diseño responsive para diferentes plataformas
